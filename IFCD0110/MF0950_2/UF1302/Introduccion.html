Introducción de HTML

¿Qué es HTML?

Es el lenguaje de marcas estándar para hacer una página web.

  • HTML significa lenguaje de marcado de Hyper Texto.
  • HTML describe la estructura de las páginas web mediante el marcado.
  • Los elementos HTML son los bloques de creación de páginas HTML.
  • HTML elements are represented by tags.
  • Los elementos HTML están representados por etiquetas.
  • Los navegadores no muestran las etiquetas HTML, pero las usan para representar el contenido de la página.

Estructura básica de un documento HTML

Ejemplo

<!DOCTYPE html>
<html lang="es-ES">
 <head>
<title>Page Title</title>
<meta charset="UTF-8">
</head>
<body>
<h1>Mi primer encabezado</h1>
<p>Mi primer párrafo.</p>
</body>
</html>

Explicación de etiquetas básicas

  1. La etiqueta <!DOCTYPE html> define la declaración HTML5 del documento.
  2. La etiqueta <html> elemento es el elemento raíz de una página HTML.
  3. El atributo lang="es-ES" especifica el lenguaje de el documento y acompaña a la etiqueta HTML.
  4. La etiqueta <head> contiene los elementos de meta información del documento.
  5. La etiqueta <meta charset="UTF-8"> define la codificación de los caracteres del documento.
  6. La etiqueta <title> espeficica el título para el documento.
  7. La etiqueta <body> contiene el contenido visible de la página.
  8. La etiqueta <h1> define el elemento "título 1" en el texto, sólo puede haber uno por documento.
  9. La etiqueta <p> define el elemento de párrafo.

Etiquetas HTML

Las etiquetas HTML son los nombres de elementos entre paréntesis angulares:

<Nombre de etiqueta> contenido va aquí ... </ tagname>

Etiquetas HTML
Normalmente vienen en pares como <p> y </p>.
La etiquetas HTML
Están formadas por una etiqueta de inicio y una segunda etiqueta de fin.
La etiqueta final
Se escribe igual que la etiqueta de inicio, pero con una barra inclinada insertada antes de que el nombre de la etiqueta.

Consejo: La etiqueta de inicio también se llama la etiqueta de apertura y la etiqueta final de la etiqueta de cierre.

Navegadores web

El propósito de un navegador web (Chrome, IE, Firefox, Safari) es leer los documentos HTML y mostrarlos.

HTML Page Structure

Below is a visualization of an HTML page structure:

<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>

Nota: Sólo el contenido dentro de la sección <body> (el área blanca arriba) se muestra en un navegador.
