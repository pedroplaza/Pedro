Formato HTML

El formato de texto, es el marcado que se aplica a un texto para añadirle datos de forma.

Al texto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido.

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios hojear sus páginas por sus títulos. Es importante utilizar las partidas para mostrar la estructura del documento.

Los títulos <h1> deben ser utilizados para los encabezados principales, seguido por títulos <h2>, <h3>, etc..., para los encabezados de menos importancia, y así sucesivamente.

Nota: Usar encabezados HTML para sólo encabezados. No utilice encabezados para que el texto GRANDE o negrita.

Etiquetas que dan formato al texto

HTML también define etiquetas especiales para marcar textos con un especial significado.

HTML utiliza elementos como <b> e <i> para el formato de salida, como negrita o cursiva texto.

Etiqueta Descripción Ejemplo Uso
<b> Define un texto en negrita Esto es un texto en negrita <b>Esto es un texto en negrita</b>
<i> Define un texto en cursiva con voz o tono alternativo al resto Esto es un texto en cursiva <i>Esto es un texto en cursiva</i>
<strong> Define un texto importante o palabras clave Esto es un texto resaltado <strong>Esto es texto resaltado</strong>
<em> Define un texto enfatizado respecto a la frase que lo contiene Esto es un texto enfatizado <em>Esto es un texto enfatizado</em>
<small> Define un texto con un tamaño más pequeño Esto es un texto pequeño Esto es un texto <small>pequeño<small>
<sup> Define un texto en superíndice Esto es superíndice Esto es <sup>superíndice<sup>
<sub> Define un texto en subíndice Esto es subíndice Esto es <sub>subíndice<sub>
<ins> Define un texto insertado con posterioridad Esto es un texto insertado con posterioridad Esto es <ins>insertado<ins> con posterioridad
<s> Define un texto inexacto o que ya no es relevante Esto es un texto inexacto e irrelevante Esto es un texto <s>inexacto</s> e <s>irrelevante</s>
<u> Define un texto subrayado Esto es un texto subrayado Esto es un texto <u>subrayado</u>
<del> Define un texto que ha sido eliminado de la publicación inicial Esto es un texto eliminado Esto es un texto <del>eliminado</del>
<mark> Define un texto remarcado simulando el rotulador amarillo. Esto es un texto remarcado Esto es un texto <mark>remarcado</mark>

Nota: Los navegadores muestran <strong> como <b>, y <em> como <i>. Sin embargo, hay una diferencia en el significado de las siguientes etiquetas: <b> y <i> define el texto en negrita y cursiva, pero <strong> e <em> significa que el texto es "importante" a efectos de su relevancia para los buscadores.
